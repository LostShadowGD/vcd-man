# vcd-man
A converter that turns any video file into Video CD formatted bin/cue files, ready for burning.
<br><br>v1.0 was made in about 8-9 hours over the span of 3 days, for no real reason... (still not sure why)

## Downloads
If you're running macOS 10.13 or 10.14, download "v1.0-5.zip" (legacy) and if you're running macOS 10.15 or higher, download "v1.0-6.zip" (main). Either version requires [Python 3.11.x](https://www.python.org/downloads/release/python-31115/) to be separately installed. The only nonstandard library used is PyQt5 or PyQt6 (in legacy or main respectively), which is bundled with vcd-man. PyQt and Qt are both released under (L)GPL.

[Download Page](https://github.com/LostShadowGD/vcd-man/releases/latest)

## Installation
After downloading the correct .zip, unzip it, open your Terminal and type `sh ` (including the space at the end), then drag the "initial-setup.sh" file into the Terminal window. Press enter, then follow the onscreen instructions.
